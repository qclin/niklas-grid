Flux Mono

It seems rather unspectacular at first sight. A small shelf with a keyboard under a wall mounted display. Letters, with unconventional, abstract shapes are moving, transforming and telling the observer that "it is never going to be the same". A blinking cursor suggests that this text can be edited. With every change; a new word, character or blank, each letter transforms into a new appearance. As soon as the user stops typing, slowly the letters are going back to their ordinary shape. The person interacting with it, have no specific influence on how the characters look. It seems to be random, but at the same time, it seems that there is a hidden system buried under the poster like aesthetics. The piece was programmed in OpenFramworks and is based a custom designed Mono Space Typeface. All elements that make a character are assigned to categories and corresponding functions to change their appearance. The program decides what function to call based on the user's text input done on a conventional keyboard. ‘Flux Mono’ explores parametric design to create a graphical, personal experience that only lives in the moment, and slowly disappeared when the interaction stops.
